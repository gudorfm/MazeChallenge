This program allows one to solve the Cretan maze challenge at https://coding-challanges.herokuapp.com/challanges/maze
To use the program you will need to register at the website
After you register enter your username and password into login.default.json and rename the file to login.json
In login.json there is an option for difficulty, the three options are BEGINNER, ADVANCED, and EXPERT
The difficulty must be one of those three and all capitalized, or the program will not function properly
